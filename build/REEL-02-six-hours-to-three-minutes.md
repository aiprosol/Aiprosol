## Common Higgsfield pitfalls + fallbacks

| Pitfall | Fallback |
|---|---|
| Faces/hands sneak in despite the prompt | Strengthen negative prompt: "no people, no hands, no faces, no humans, no skin." Re-roll up to 3×. If still bad, use a stock-footage cutaway (Storyblocks / Pexels) for that single shot. |
| Split-screen output drifts off-center | Generate the two halves as separate shots, composite in CapCut with a manual 50/50 split layout. |
| Violet renders too blue or too pink | Add hex codes explicitly: "violet color #8B5CF6 primary, #C084FC accent." If still off, color-grade in post with a violet LUT. |
| In-shot text comes out garbled (common) | **Never trust Higgsfield for in-shot text.** Render the shot blank, add the overlay in CapCut/Premiere using Space Grotesk 800. |
| Shots return ~5s but you need 3.5s | Trim in post. Always render long, trim down. |
| Brand orb doesn't match Reel #1's orb | **Reuse Reel #1's orb clip** when you can. Series continuity > novel render. |
